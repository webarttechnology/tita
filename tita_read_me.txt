======= FRONT END ========

1> All Pages link Dynamic ->
2> Register Menu -> User Registration and Installer Register
3> User Registration will store data into User Table in database.
4> Installer Registration will store data into Installers Table in database and Images will store in Imges/Users.
4> Quote will store data into Quote Table in database.


======= Admin ========

1> User Registration from Front end will show in Admin Panel, admin can edit and delete Users. 
2> Installer Registration from Front end will show in Admin Panel and admin can view their details on Action. 
3> Installer Report from Front end will show in Admin Panel POC(view) . 
4> Quote Registration from Front end will show in Admin Panel.
